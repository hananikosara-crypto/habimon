# HabiMon — 習慣×モンスター育成アプリ

## 技術スタック

| 分類 | 技術 |
|------|------|
| フレームワーク | Next.js App Router |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS + shadcn/ui |
| バックエンド | Supabase (DB + Auth) |
| 状態管理 | Zustand |
| デプロイ | Vercel |

## ディレクトリ構成

```
src/
├── app/          # Next.js App Router ページ・レイアウト
├── components/   # 再利用可能な UI コンポーネント
│   └── ui/       # shadcn/ui コンポーネント
├── lib/          # ユーティリティ・Supabase クライアント等
├── hooks/        # カスタム React フック
├── stores/       # Zustand ストア
├── types/        # TypeScript 型定義
└── constants/    # 定数・設定値
```

## コーディング規約

- コンポーネントは `function` 宣言 + `export default` を使用
- コメントは日本語推奨
- ファイル名はケバブケース（例: `habit-card.tsx`）
- 型定義は `src/types/index.ts` に集約

## 重要な仕様

### モンスターと目標の対応
- 目標（Goal）1つにモンスター1体が紐づく
- 目標を達成・継続するとモンスターが成長する

### 進化ステージ

| ステージ | 名称 |
|---------|------|
| 1 | たまご |
| 2 | ベビー |
| 3 | チャイルド |
| 4 | ジュニア |
| 5 | アダルト |
| 6 | エボリューション |
| 7〜11 | 最終形態（コースにより異なる） |

- **1年コース**: 7段階
- **3年コース**: 11段階

### ポイント設計

| アクション | 獲得ポイント |
|-----------|------------|
| 習慣完了 | +10pt |
| 7日ストリーク | +20pt（ボーナス）|
| 30日ストリーク | +50pt（ボーナス）|

### 罰しない設計
- サボってもモンスターは死なない・弱体化しない
- ポジティブな強化のみでモチベーションを維持する設計
